00016 /*

00018     That way, no matter whether you use Carbon's FNNotify/FNSubscribe, BSD's
00019     kqueue or whatever, the object being notified can react to change
00020     notifications the same way, and you can easily swap one out for the other
00021     to cater to different OS versions, target volumes etc.
00022 */
